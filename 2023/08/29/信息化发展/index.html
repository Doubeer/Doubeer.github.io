# 信息系统

# 信息系统

定义：

​ 输入数据，进行加工，产生信息的系统

显著特点：

​ 面向管理和支持生产

组成部分：

​ 硬件、软件、数据库、网络、存储设备、感知设备、外设、人员以及把数据处理成信息的规程

分类（用途类型划分）：

​ 电子商务系统、事务处理系统、管理信息系统、生产制造系统、电子政务系统、决策支持系统

# 信息系统的生命周期

四大：立项、开发、运维、消亡

  1. 立项（系统规划）：写可行性报告
  2. 开发：
    1. 系统分析：提出逻辑模型
    2. 系统设计：提出物理模型
    3. 系统实施：将设计付诸实施
  3. 系统运行和维护：维护和评价，记录系统运行情况
  4. 消亡

5 小：系统规划、系统分析、系统设计、系统实施、运行维护

# 工业互联网（掌握）

是第四次工业革命的重要基石

# 工业互联网的四大层级

  1. 网络是基础：工业互联网网络体系有三部分
    1. 网络互联
    2. 数据互通
    3. 标识解析
  2. 平台是中枢：平台体系包括边缘层、IaaS、PaaS、SaaS，作用如下
    1. 数据汇聚
    2. 建模分析
    3. 知识复用
    4. 应用创新
  3. 数据是要素：工业互联网有三个特性
    1. 重要性
    2. 专业性
    3. 复杂性
  4. 安全是保障：工业互联网安全又三大特点
    1. 涉及范围广
    2. 造成影响大
    3. 企业防护基础弱

# 工业互联网六大典型应用模式

  1. 平台设计
  2. 智能化制造
  3. 网络化协同
  4. 个性化定制
  5. 服务化延申
  6. 数字化管理
